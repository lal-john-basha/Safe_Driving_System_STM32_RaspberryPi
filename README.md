# 🚗 Safe Driving System Using STM32 & Raspberry Pi

The **Safe Driving System** is designed to improve road safety by detecting **driver alcohol consumption and monitoring driver behavior**.

This system combines **embedded systems and computer vision** using **STM32 microcontroller** and **Raspberry Pi** to detect unsafe driving conditions and trigger alerts.

The goal of this project is to **prevent drunk driving and enhance driver safety**.

---

# 📌 Project Overview

Road accidents caused by **drunk driving** are a major safety concern.

This project integrates multiple hardware components and sensors to monitor the driver's condition in real time.

The system works by:

1. Detecting alcohol using an **MQ3 alcohol sensor**
2. Monitoring the driver using a **Raspberry Pi camera**
3. Sending warning alerts if unsafe conditions are detected
